### 框架说明
本框架基于SpringBoot 2.6.7 构建，集成了Mysql 8.0、Redis、MyBatis、FastJSON、
Druid数据源、Lombook等常用插件，可作为一个快速上手的开发框架。

启动前，请修改application.yml中Mysql、Redis的地址。

使用mybatis generator前，请修改generatorConfig.xml中的数据库地址。
